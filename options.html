Image Autosizer by k-mouse

Quick guide

Open up an image file to use this extension. Try it on Wikipedia: Picture of the Day (click the image twice to open the image file).

  • The picture should be resized to fit your browser window
  • You can click on the picture to view it in its original size, and drag it around with your mouse
  • Double-click the picture to view it in fill mode
  • Press F11 to view the picture in full screen, looks especially good with black background
  • Press c to temporarily change background color

Press shift+c to temporarily enable checkerboard background.

Inertial panning

    Enabling Inertial panning will make the image continue sliding for a little while after letting go of the mouse button. Also known as smooth scrolling.

    • Inertial speed is the speed at which the window will scroll after letting go of the mouse button.

    • Inertial deceleration determines how fast the panning comes to a stop after letting go of the mouse button.

    • Only the panning events for the last milliseconds before letting go of the mouse button is used for the inertial speed calculations. The number of milliseconds is given by recording time.

Miscellaneous

    Show image info text pops up a small caption in the bottom right corner of the image with information about the image resolution and the current view mode.

    • Duration specifies how long the image info text stays visible.

    • Close image on long press. Holding your left mouse button down for more than a second will close the page with the image.

    • Always use pointer cursor will prevent the zoom-in and zoom-out cursors from showing up which can be unintuitive when extra display modes are enabled.

    • Checkerboard background for transparency will display a checkerboard behind the image such that transparency will be easily identifiable.

Display modes control how the image is displayed and fitted to the window.

Enabled modes will be cycled through when left-clicking on the image.

Default mode is the first mode the image will enter when opening up.

The selected Double-click mode will be activated when double-clicking on the image. This mode can be a disabled mode.
